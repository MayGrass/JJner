### 目前包含

- Facebook、Youtube、歌單等等相關資訊連結
- JJ留言紀錄
- 自訂新增抓哪台的JJ留言
- 自動點擊"國王的金幣"額外獎勵，其它台同時也通用
- 揪團後自動返回原實況台
- News跑馬燈自動更新YT最新影片、Twitch昨日實況摘要
- 新增實況日報網站連結，點擊Twitch News按鈕即可前往觀看
- 自動檢測圖奇播放器發生錯誤按鈕並點擊重整
- ~~LoL戰績摘要~~(功能壞光光3.0.0版本已移除)
- ~~圖奇開台通知(含GodJJ鈴聲)~~ (2.3.2版移除此功能)
